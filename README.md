# Speech Text Reader

A simple web app intended for people with speech impairement that utilizes the Speech Synthesis Web API to facilitate the transformation of text to speech. It includes twelve baked in expressions, specifically tailored to younger aged users, that activate by clicking their corresponding box areas. To read out custom text, click on the toggle text box button to input your custom text in the text area. Users can select the specific voice and language of the speech synthesizer on the relevant select element of the text box.

Made with:

- HTML, CSS, vanilla JavaScript
- Leveraging of the text-to-speech capabilities of the Speech Synthesis API of the browser
